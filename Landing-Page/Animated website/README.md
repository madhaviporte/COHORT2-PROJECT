# 🚀 Animated React Website

This is a fully frontend animated website built using React. The main goal of this project was to improve my frontend skills and explore smooth animations, routing, and responsive design.

## 🔗 Live Demo

👉 https://animated-website-green.vercel.app/

## 📌 About the Project

I created this project to practice building modern UI with animations and better user experience. The website includes multiple sections with smooth navigation using React Router. I focused on keeping the design clean, interactive, and responsive across all devices.

## ⚙️ Tech Stack

* React.js
* Tailwind CSS
* React Router DOM
* JavaScript (ES6+)
* HTML5
* CSS3

## ✨ Features

* Smooth animations across the website
* Fully responsive design (mobile, tablet, desktop)
* Multi-page navigation using React Router
* Clean and modern UI
* Fast loading frontend

## 📱 Responsiveness

The website is fully responsive and adapts well to different screen sizes. It works smoothly on mobile devices, tablets, and desktops.

# Go to project folder

cd your-repo-name

# Install dependencies

npm install

# Run the project
